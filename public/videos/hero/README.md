# Hero Video Slot

Drop a 5-15 second muted MP4 here as `hero-loop.mp4`.

Recommended specs:
- Format: MP4 (H.264)
- Dimensions: 1920×1080 or 1280×720
- Duration: 5-15 seconds (it's looped)
- Audio: muted (browser auto-plays muted videos only)
- Theme: dark/moody soccer footage — close-up of jersey, stadium b-roll, ball shots
- Size: keep under 5 MB for fast loading

Free CC0 sources:
- https://www.pexels.com/search/videos/soccer/
- https://pixabay.com/videos/search/soccer/
- https://www.coverr.co/

Until the file is here, the gradient fallback in HeroVideo.tsx handles the section gracefully.
